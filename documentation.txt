===================
3. CUSTOMIZATION
===================
Profile Information:
- Name: Update in header section
- Title: Update in header section
- About: Modify text in about section
- Photo: Replace image URL

Skills:
- Add new skills in grid layout
- Format: <div class="bg-white p-4 rounded-lg">
- Maximum 6 skills recommended per row

Experience:
- Format for each entry:
  * Title
  * Company
  * Date
  * Description
- Use border-l-4 for left border style

Education:
- Format for each entry:
  * Degree
  * Institution
  * Year
- Maintain consistent spacing

Contact:
- Update email
- Update phone
- Update social links
- Maintain icon alignment

===================
4. TROUBLESHOOTING
===================
Common Issues:

Images not loading:
- Check image URL
- Verify internet connection
- Ensure proper image format

Animations not working:
- Check AOS script loading
- Verify data-aos attributes
- Clear browser cache

Layout issues:
- Check viewport settings
- Verify Tailwind CSS loading
- Test responsive breakpoints
